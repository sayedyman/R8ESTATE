# AI Workflow

Always follow this workflow before writing any code.

## Step 1
Understand the task completely.

Do not start coding immediately.

## Step 2
Read the project context.

Read:

- PROJECT_CONTEXT.md
- DECISIONS.md

## Step 3
Break the task into small steps.

Explain the plan before implementation.

## Step 4
Implement one feature only.

Avoid touching unrelated files.

## Step 5
Run a self review.

Check

- readability
- accessibility
- responsive design
- consistency
- performance

## Step 6
If something is unclear, ask before coding.
